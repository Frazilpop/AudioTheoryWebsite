FAQ

When purchasing AudioTheory software (either directly or through Steam) do I get access to both the Mac and PC versions?

Yes. Please note however that the Mac App Store version is purchased through Apple and only works on Mac.
I seem to be missing a feature detailed in the User Guide?

New features have been added through post-launch updates. You probably need to update to the latest version.
I'm left handed, will AudioTheory software still be of use to me?

I'm left handed too. As such, left handed support is a big focus of mine. You can choose left handed mode in the settings and all features and diagrams will display accordingly.
How do I receive updates?

If you purchased directly then you can download the latest version from your Gumroad account. For Steam users, new updates can be downloaded through the Steam client. For Mac App Store purchases the application can be updated from the macOS App Store.
These updates are free, right?

Yes, all updates and new features are (and always will be) free for all existing customers.
Is the Mac version 64 bit?

Yes – this means it will work on versions of macOS that only support 64 bit applications. We are also a registered Apple Developer and all AudioTheory releases has been notarised – this means the file contents is verified by Apple and our apps comply with the security procedures introduced in Mac OS Catalina.
I have a question not listed here.

No worries, drop me an email: fraser@audiotheory.co.
